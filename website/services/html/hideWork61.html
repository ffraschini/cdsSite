Cerego

We developed this iOS app from scratch and worked on the Android version. Cerego is the best way to learn faster and remember longer. Discover hundreds of courses, build your own, and easily improve memory performance as you learn.

Technologies

iOS version: Mathjax, AppsFlyer, CorePlot, Restkit, FacebookSDK, Accounts, Social, AFNetworking, FTUtils, FlurrySDK, SOCKit, TestflightSDK, TransitionKit, AdSupport, MediaPlayer, MessageUI, Security, StoreKit, CoreText, AudioToolbox, AVFoundation, QuartzCore, CFNetwork, MobileCoreServices, Foundation, CoreData, CoreGraphics, UIKit. Android version: Java + Android SDK, AndroidAnnotations, ORMlite ,Volley, SQLite, JUnit for Android, Intellij IDEA 13, Genymotion
