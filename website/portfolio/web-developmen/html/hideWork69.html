Fox

Project to manage human resources and the job performed by a theater maintenance company. It’s a CRUD project customized to handle stuff like: Who’s part of each job, which tasks should be done for each job, which vehicles will be used in each job, the time worked by each employee, the expenses for each job. Also there is a message center included when leaders can send messages to each employee. Files can be uploaded by admins and employees can view them.

Technologies

Ruby on Rails, Javascript, HTML, CSS
