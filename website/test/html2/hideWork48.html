JSCAPE

This application helps users easily manage, view, transfer and share files. It is the mobile version of the existing JSCAPE MFT Server. The main features included are: uploading images and videos from camera and gallery, previewing common file formats (PDF, Word, Excel, PowerPoint, JPG, MP3, etc), sharing files with customers or co-workers, and organizing frequently accessed files using tag clouds - among others.

Technologies

iOS: XCode, Objetive-C, AFNetworking, QuickLook Framework, AddressBook Framework, Security Framework, SQLite.
Android: Eclipse, Android SDK, Java, ActionBarSherlock, aFileChooser, SQLite.
