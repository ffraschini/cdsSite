MiJay

Mijay is a music social networking application that allows you to view and share the music your friends are listening to on their iPhone and iPod touch in real-time. The client trusted us with doing the maintenance and bug fixing of this application. We also added some new features and changed all the images for high resolution images so it supports the new iPod and iPhone 4 Retina displays.

Technologies

IOS SDK, XCode Interface Builder, SQLite, Facebook API , Twitter Rest Api and IOS Frameworks: MPMusicPlayerController, Accounts (iOS 5),Twitter (iOS 5), CoreData Framework, Accounts Framework, Facebook iOS SDK, MediaPlayer Framework, Client/server, MessageUI Framework, AddressBook Framework, Push Notifications, Syncs device's music with server.
