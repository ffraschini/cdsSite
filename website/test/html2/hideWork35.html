iGrade Soccer

Rated four stars on iSoccerApps.com, this application gives soccer coaches detailed and precise information on Players, Practices, Games, Attendance and More. The service we provided was maintenance, enhancement, and bug fixing. We added audio notes, fixed a database bug as well as a problem that was occurring with the notes feature, and implemented some design changes too.

Technologies

MessageUI Framework, MediaPlayer Framework, CoreData Framework, CorePlot, Reuse the same code for multiple versions.
