Skout for iPhone

We worked on the iPhone versions of this widely known (more than two million users) network application for meeting new people and having fun. We have been developing client-side using Objective-C, mainly working on the GUI logic. Some of its remarkable features are: integrated chat, location based, and in-app points.

Technologies

AdMob, Tapjoy, CoreLocation Framework, AddressBook Framework, Client/server, Push Notifications.
